# cat-wheels

Cat Wheels is a prototype game where you control a cat riding a skateboard.  
It was developed as one of my first game development projects at the age of 11.

Built with Unity.

## Play the Game
You can play or download the game here:  
https://randmsdev.itch.io/cat-wheels

## Features
- Basic character movement
- Simple game mechanics

## What I Learned
- Fundamentals of game logic
- Basic game development using Unity
